# Naruto-Game-Officiel-Repo

A fan-made Naruto fighting game for J2ME mobile phones.

## Features

* Play as **6 different Naruto characters**
* Each character has a **unique Special Attack**
* Build chakra during battle
* Use powerful techniques and ultimate attacks

## Controls

| Key   | Action                                       |
| ----- | -------------------------------------------- |
| **0** |  Attack unlimited sp                                |
| **7** | Special Attack (Requires 50% Chakra or more) |
| **9** | Ultra Attack (Requires Full Chakra)          |

## Playable Characters

* Naruto Uzumaki
* Sasuke Uchiha
* Sakura Haruno
* Kakashi Hatake
* Naruto Rikudoo
* Sasuke Rinnesharigan

## Gameplay

Fight against your opponents, charge your chakra, and unleash devastating ninja techniques. Manage your chakra wisely to activate powerful Special and Ultra attacks.

## Disclaimer

This is a fan project inspired by the Naruto series. Naruto and all related characters belong to their respective owners. This game is not affiliated with or endorsed by the rights holders.

### J2ME Requirements

* MIDP 2.0
* CLDC 1.1
* Screen Resolution: 240×320 or higher
* Java-enabled mobile phone or emulator
